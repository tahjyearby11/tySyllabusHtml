Course Information

  • Course:Software Enginerring (CPSC-44000)

  • Course Credit Hours:3

  • Course Description: Methods, strategies, and tools for implementing software systems, particularly as part of a development team. Topics include the software development life cycle, Unified Modeling Language, software testing techniques, software security, open-source development, requirements gathering and documentation, maintenance, and basic software project management.

    Prerequisites: CPSC-24500 Object Oriented Programming
